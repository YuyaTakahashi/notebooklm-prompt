# NotebookLM Prompt Inserter

NotebookLMのプロンプト入力欄に、用途別のプロンプトをワンクリックで挿入するChrome拡張機能です。

## 機能

### データ・提案系
- **📊 数字系の可視化資料** — グラフやチャートを含む資料作成用
- **📖 業務マニュアル作成** — 手順書や説明資料の生成
- **💡 企画・提案用スライド** — ビジネス提案のスライド作成

### 思考・解釈系
- **🗺️ 図解構成の作成** — 複雑な概念を図で説明
- **🌱 わかりやすい解説** — 初心者向けの簡潔な説明
- **📚 書籍要約（スライド分割）** — 本の内容をスライド形式で分割要約

## 使い方

1. NotebookLMの入力欄にフォーカス
2. 拡張機能アイコンをクリック
3. 目的のボタンを選択
4. プロンプトが入力欄に自動挿入される

入力欄が見つからない場合、プロンプトはクリップボードにコピーされます。

## インストール

1. `manifest.json`がある状態で、Chrome拡張機能の開発者モード（chrome://extensions）から読み込む

## ファイル構成

```
notebooklm-prompt/
├── manifest.json      # 拡張機能の設定
├── popup.html         # ポップアップUI
├── popup.js           # プロンプト挿入ロジック
└── prompts/           # 用途別プロンプト
    ├── viz.js         # 可視化資料用
    ├── manual.js      # マニュアル作成用
    ├── proposal.js    # 提案用
    ├── diagram.js     # 図解作成用
    ├── easy.js        # わかりやすい解説用
    └── book.js        # 書籍要約用
```

## 技術仕様

- **Manifest**: V3
- **対象ブラウザ**: Chrome/Chromium系
- **権限**: `activeTab`, `scripting`
